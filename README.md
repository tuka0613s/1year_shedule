# 1-Year Schedule Planner

1画面で1年間の予定を俯瞰・管理できる、モダンでプレミアムなスケジュール管理アプリです。

## 🚀 公開URL
https://tuka0613s.github.io/1year_shedule/

## ✨ 特徴

- **Googleカレンダー双方向同期**:
  - Google Calendar APIと連携し、予定の表示だけでなく**アプリからの追加・編集・削除**が可能。
  - ページ読み込み時に自動で最新の予定を同期。
  - セッション内での認証維持により、ページ更新時の再ログイン頻度を低減。
  - アプリのカラーとGoogleカレンダーのカラー（ColorID）を相互にマッピング。
- **重複予定の柔軟な表示**: 
  - **スタッキング表示**: デフォルトではセル内で上下に並べて表示。
  - **横並び（並列）表示**: PC等の広い画面で、予定を横に分割して表示可能（チェックボックスで切り替え）。
- **更新ボタン (↻)**: ページ全体をリロードし、PWA（ホーム画面追加時）でも確実に最新のコードとデータを取得。
- **スマート・レスポンシブ**:
  - PCでは1画面に収まる最適なグリッド表示。
  - スマートフォン（iPhone含む）では横スクロールで全月を閲覧可能。
  - モーダルがビューポート幅に合わせて調整され、横画面（ランドスケープ）でもスクロールして入力しやすいUI。
  - iOS のズームを防ぐためフォントサイズを 1rem 以上に設定。
- **超圧縮モード**: チェックボックス一つで31行を画面内に強制フィットし、年全体を1画面で完全表示。
- **連日予定の視覚化**: 複数日にわたる予定を連結バー形式で表示。
- **動的な年切り替え**: 前後の年にワンクリックで移動可能（初期表示は起動時の実年）。
- **週番号表示**: 必要に応じて週番号（W1〜W53）を表示。
- **スケジュール編集**: クリック一つで予定の追加・編集・削除・カラー変更が可能。
- **PWA対応**: iPhoneやAndroidのホーム画面に追加して、アプリのように利用可能。
- **ローカル保存**: LocalStorage でブラウザを閉じても予定が保持される（Google連携しない場合）。

## 📝 使い方

1. **予定の追加**: 日付セルをクリックするとモーダルが開き、タイトル・期間・カラーを設定して保存。
2. **Googleカレンダー連携**: 
   - 右上の「Google連携」ボタンをクリック。
   - 初回は設定画面（右クリックでいつでも開けます）でAPIキーとクライアントIDを入力。
   - モーダル内の「Googleカレンダーに保存する」にチェックを入れると、Googleカレンダー側に同期されます。
3. **予定の編集・削除**: 既存の予定バーをクリックして編集モーダルを開く。Googleの予定も直接編集・削除可能です。
4. **週番号の表示**: ヘッダーの「週」チェックボックスで週表示を切り替え。
5. **横並び表示**: ヘッダーの「横」チェックボックスで、重複予定を横並びに分割。
6. **圧縮モード**: ヘッダーの「圧縮」チェックボックスで全31行を1画面に強制フィット。
7. **アプリの更新**: 「↻」ボタンをクリックして最新状態にリロード。
8. **年の切り替え**: ヘッダーの `<` / `>` ボタンで前年・翌年に移動。
9. **今日へ移動**: 「今日」ボタンをクリックすると今日の行がフラッシュハイライトされる。
10. **テーマの変更**: ヘッダーのテーマアイコン（🌙 / ☀️ / ✨）をクリック。

## 📱 モバイル対応 (iPhone)

- フォント 1rem 以上でピンチズーム防止。
- 開始日・終了日の入力欄を縦並びに変更し、タップしやすく。
- モーダルはビューポート幅 - 32px で自動調整。
- 横画面（高さが低い）時はモーダル内スクロールが有効になり、すべての項目にアクセス可能。
- グリッドは `-webkit-overflow-scrolling: touch` による慣性スクロール対応。
