# 1-Year Schedule Planner

1画面で1年間の予定を俯瞰・管理できる、モダンでプレミアムなスケジュール管理アプリです。

## 🚀 公開URL
https://tuka0613s.github.io/1year_shedule/

## ✨ 特徴
- **1画面集約レイアウト**: 12ヶ月分を横一列に並べ、1年をひと目で把握可能。
- **プレミアム・デザイン**: ダークモードとグラスモーフィズムを採用したスタイリッシュなUI。
- **スマート・レスポンシブ**: PCでは1画面、スマホ（縦）では横スクロール、スマホ（横）では最適化されたグリッド表示。
- **連日予定の視覚化**: 複数日にわたる予定を連結して表示。
- **日本の祝日対応**: 日本の祝日を自動取得し、赤色で強調表示。
- **動的な年切り替え**: 前後の年にワンクリックで移動可能。
- **週番号表示**: 必要に応じて週番号（W1〜W53）を表示。
- **スケジュール編集**: クリック一つで予定の追加・編集・削除・カラー変更が可能。

## 🛠 テクノロジー
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- GitHub Pages (Hosting)

## 📝 使い方
1. 日付のセルをクリックすると予定追加モーダルが開きます。
2. 既存の予定をクリックすると編集・削除ができます。
3. 右上の「週番号」チェックボックスで週表示を切り替えられます。
4. 今日の日付は青いハイライトで表示されます。
